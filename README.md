Sekalaisia Ohjelmia
-------------------

-ToDo: v1.0:
       Ohjelma jonka avulla voit helposti tehdä listan työpöydän yläkulmaan.
       Ohjelma ymmärtää pikakomennot alt+s (submit), alt+r (ready) ja alt+l(load).
       Lista tunnistaa checkatut komponentit ja kun kaikki on checkattu lista sulkeutuu.

       v1.2
       Lista muistaa edellisen listan. Edellinen lista haetaan .txt-tiedostosta.
       
       v1.3:
       Listaa voi nyt muokata. Painamalla edit-nappia listan yläpuolella aukeaa .txt
       tiedosto jossa voi muokata listaa tai lisätä asioita sinne. Muokkauksen jälkeen
       muista tallentaa .txt tiedosto (ctrl+S) ja sen jälkeen paina load-nappia. Uusi
       lista lataa tällöin.
